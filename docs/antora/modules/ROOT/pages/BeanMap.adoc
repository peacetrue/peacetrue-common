= BeanMap

`BeanMap` 是 `LinkedHashMap<String, Object>` 的子类，
表示一个泛化的 Java Bean 对象。

如果说一个方法接收 `BeanMap` 作为参数，
实际上表示它可以接收 `Map<String, Object>`，
`BeanMap` 的好处在于可以避免泛型转换。
